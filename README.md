# ARCT | Architectural Portfolio

Professional portfolio website for an architectural designer, built with a focus on high performance, clean geometry, and seamless user experience. Developed to demonstrate modern frontend practices and a deep understanding of architectural aesthetics.

## 🚀 Technical Stack

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animation**: Framer Motion
- **Architecture**: Feature-Sliced Design (FSD) principles
- **Deployment**: Vercel

## 🏗️ Key Features

- **Performance-First**: Optimized for "green zone" scores on PageSpeed Insights using Next.js image optimization and server-side rendering.
- **Dynamic Routing**: Implementation of a scalable portfolio system with dynamic page generation.
- **Interactive UI**: Fluid animations and layout transitions to maintain a premium feel.
- **Responsive Design**: Fully adaptive layout across all devices, including specialized touch-friendly navigation for mobile.
- **Professional Aesthetics**: Minimalist design language, focused on typography and structural clarity.

## 🛠️ Development

This project is built with maintainability in mind. It follows a modular structure where UI components, business logic, and data are decoupled.

```bash
# Clone the repository
git clone [https://github.com/Recordline1/ARCT.git](https://github.com/Recordline1/ARCT.git)

# Install dependencies
npm install

# Run the development server
npm run dev
